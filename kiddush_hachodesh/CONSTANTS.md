# קבועים אסטרונומיים מהלכות קידוש החודש (רמב"ם)

מודול `kiddush_hachodesh` מחשב קבועים אסטרונומיים **בחישוב אלגברי מדויק**
(`fractions.Fraction`, ללא נקודה צפה) הנגזרים משני נתונים מוצהרים במפורש
בהלכות קידוש החודש, פרק ו':

1. **עונת המולד** (6:3) — משך החודש הסינודי הממוצע: כ"ט יום, י"ב שעות,
   תשצ"ג חלקים (793 חלקים; חלק = 1/1080 שעה). בחלקים שלמים: 765,433 חלקים —
   מספר שלם, מדויק, ללא קירוב.
2. **מחזור המעברות בן 19 השנים** (6:11-13) — 235 חודשים במחזור, מהם 7
   שנים מעוברות (שנים ג', ו', ח', י"א, י"ד, י"ז, י"ט).

משני הנתונים המדויקים האלה, ובלי להוסיף שום הנחה חדשה, ניתן **לגזור
אלגברית** קבועים אסטרונומיים נוספים שהרמב"ם עצמו דן בהם בהמשך ההלכות
(פרקים י"א-י"ב, מהלכות החמה והלבנה):

| קבוע נגזר | נוסחה | ערך מדויק (שבר) | ≈ | ערך מודרני |
|---|---|---|---|---|
| שנת חמה משתמעת | 235 × עונת-המולד ÷ 19 | `35975351/98496` יום | 365.2468222 יום | 365.2421900 יום |
| מהלך יומי של החמה | 360° ÷ שנת-החמה המשתמעת | — | 0°59′8.29″ | 0°59′8.33″ |
| התרחקות יומית של הלבנה מהחמה | 360° ÷ עונת-המולד | — | 12°11′26.69″ | 12°11′26.70″ |
| מהלך יומי של הלבנה (ביחס לכוכבים) | התרחקות + מהלך-החמה | — | 13°10′34.97″ | 13°10′34.89″ |

שווה לב: הערכים המחושבים כאן (0°59′8″ לחמה, 13°10′35″ ללבנה) תואמים כמעט
בדיוק את המספרים שהרמב"ם עצמו נותן במפורש בפרקים י"א-י"ב לתנועה הממוצעת של
המאורות — למרות שהם נגזרו כאן *רק* מעונת המולד ומבנה מחזור 19 השנה, בלי
להיזקק לטבלאות המהלך הממוצע בפני עצמן. זה חיזוק אלגברי לעקביות הפנימית של
המערכת ההלכתית.

## שגיאת הסחיפה (הפרש בין השנה המשתמעת לשנה הטרופית האמיתית)

השנה בת 19-השנה המשתמעת ארוכה מהשנה הטרופית האמיתית (המודרנית) בכ-0.00463
יום לשנה (כ-6.67 דקות). המשמעות: הלוח העברי הקבוע (כפי שקבעו רמב"ם) סוטה
כ-יום שלם ביחס לעונות השנה מדי **כ-216 שנה** — עובדה ידועה, המחושבת כאן
ישירות מהקבועים ההלכתיים ולא מצוטטת כנתון חיצוני.

## שימוש

```bash
python -m kiddush_hachodesh.cli constants   # קבועי-יסוד + קבועים נגזרים
python -m kiddush_hachodesh.cli drift       # השוואה מדויקת לערכים מודרניים
python -m kiddush_hachodesh.cli molad 5786  # מולד תשרי לשנה עברית נתונה
```

```python
from fractions import Fraction
from kiddush_hachodesh.mean_motion import implied_mean_tropical_year_days

year: Fraction = implied_mean_tropical_year_days()  # 35975351/98496
```

## מקורות

- רמב"ם, משנה תורה, ספר זמנים, הלכות קידוש החודש, פרק ו' (יסודות עונת
  המולד ומחזור המעברות) ופרקים י"א-י"ב (מהלכי החמה והלבנה).
- ערכים מודרניים להשוואה: `mean_motion.MODERN_*` — ממוצעים אסטרונומיים
  נמדדים (IAU / אלמנך אסטרונומי), לא הלכתיים, מובאים כאן רק כרפרנס להשוואה.

## היקף ומגבלות

המודול הזה מכסה את שכבת ה"ארבעה מהלכים" הבסיסית (עונת המולד, מחזור
המעברות, ומהלכי החמה/לבנה הממוצעים הנגזרים מהם) ואת חישוב המולד עצמו
(`molad.py`, אריתמטיקת חלקים שלמה). הוא **אינו** ממדל את התיקונים
הטריגונומטריים המלאים של מסלול הלבנה (מסילה, ראש, קו-רוחב, ראיית הירח —
פרקים י"ג-י"ט), שהם נושא נפרד ומורכב בהרבה.
