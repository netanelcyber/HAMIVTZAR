Premium Analytics dashboard for Jetpack sites.

== Description ==

Provides a premium analytics dashboard for Jetpack-connected WordPress sites.

== Installation ==

1. Install the plugin through the WordPress plugins screen.
2. Activate the plugin.
3. Navigate to the Analytics menu to view the dashboard.
